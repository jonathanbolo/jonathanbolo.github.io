Sublime Text shortcuts

Here are some shortcuts that I use in Sublime Text 2. I will add more as I learn and practice them.

Shortcut keysAction
Command + DSelect word
Command + LSelect Line
Command + Shift + SpaceSelect scope
Control + Shift + MSelect within brackets
Command + Shift + ASelect within tags
Command + Shift + VPaste and Indent
Command + ]Indent line
Command + [Unindent line
Control + Shift + KDelete current line
Control + KDelete to End
Command + DeleteDelete to beginning
Command + JJoin lines
Command + /Toggle comment
Command + Shift + EnterInsert Line before
Command + EnterInsert Line after
Command + Alt + .Close tag
Control + Command + GSelect all occurrences of current selection
Control + MJump to matching bracket
