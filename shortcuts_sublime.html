Sublime Text shortcuts

Here are some shortcuts that I use in Sublime Text 2. I will add more as I learn and practice them.

Shortcut keysAction
Command + XCut line
Command + LSelect Line
Command + DSelect word
Control + Command + GSelect all occurrences of current selection
